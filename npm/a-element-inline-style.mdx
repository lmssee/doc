## 使用

## 使用

设置行内样式

```ts
import { setStyle } from 'a-element-inline-style';

setStyle(document.body, {
  width: '100%',
  '--custom-width': '80%',
});
```

移除行内样式

```ts
import { removeStyle } from 'a-element-inline-style';

removeStyle(document.body, ['width', '--custom-width']);
```

属性命名请使用*小驼峰*
