# 插入排序

给定的一组数据，初识时假设第一个记录自有一个序列，其余的无序。从第二个开始，按照大小依次进行处理插入到有序序列中，直至最后一个有序的为止。

```js
function insertSort(arr) {
  let tem;
  for (let i = 1, length = arr.length; i < length; i++) {
    tmp = arr[i];
    for (let j = i - 1; j >= 0; j--) {
      if (tmp < arr[j]) arr[j + 1] = [arr[j], (arr[j] = tmp)][0];
      else break;
    }
  }
  return arr;
}
```
