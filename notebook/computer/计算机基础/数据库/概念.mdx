# 概念

## 数据

数据（ Data ）是用于描述客观事物的各种物理符号，包括数值、文字、图形、图像、声音以及由若干个数据项组成的数据元素等。它是数据库中存储和处理的基本对象。

## 数据处理

数据处理（ Data
Processing ）是指对数据进行采集、存储、加工和传输等的一系列活动的总和。数据处理目的是为了利用计算机从大量繁杂的原始数据中提炼出对人们有用的信息，以便于人们利用这些信息做出科学的决策。同时也可以利用计算机科学地存储和管理信息。

## 数据库

数据库（ Data Base ， DB ）简单地说是存放数据的仓库。只不过这个仓库是按照某一种数据模型在计算机内建立起来的，数据存储的目的是为了实现数据共享。因此，所谓数据库是指长期存储在计算机内的、有组织的、可共享的数据集合。

## 数据库管理系统

数据库管理系统（ Data Base Management
System ， DBMS ）是数据库系统的一个重要组成部分，是数据库系统的核心。在计算机内有了数据库之后，还需要有一个管理软件，才能实现对数据的管理、维护和使用。完成这个任务的是一个系统软件，称之为数据库管理系统。数据库管理系统是位于用户与操作系统之间的一层数据管理软件，主要功能包括以下几个方面：

- 数据定义功能。 DBMS 提供数据定义语言（ Data Definition Language ， DDL ），用户通过它可以方便地对数据库中的数据对象进行定义，如定义数据库、基本表、视图等;
- 数据操纵功能。 DBMS 还提供数据操纵语言（ Data Manipulation Language ， DML ），用户可以使用 DML 操纵数据实现对数据库的基本操作，如查询、插入、删除和修改等;
- 数据库的运行管理。数据库在建立、运行和维护时由数据库管理系统统一管理、统一控制，以保证数据的安全性、完整性、多用户对数据的并发使用及发生故障后的系统恢复;
- 数据库的建立和维护功能。它包括数据库初始数据的输入、转换功能，数据库的转储、恢复功能，数据库的重组织功能和性能监视、分析功能等。这些功能通常是由一些实用程序完成的;

## 数据库系统

数据库系统（ Data Base System ， DBS ）是指在计算机系统中引入数据库后的系统，一般由数据库、数据库管理系统（及其开发工具）、应用系统、数据库管理员和用户构成。数据库管理员（ Data Base
Administrator ， DBA ）主要负责全面管理和控制数据库系统。
