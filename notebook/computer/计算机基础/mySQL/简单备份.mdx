# 导出为文本

```sql
select * from main
into outfile "";
```

## 导出为数据

```bash
# 导出所有数据
mysqldump -u root -p --all-databases > all.tex
# 导出所有数据 test 库
mysqldump -u root -p --all-databases > all.tex
# 导出 test 库中的 main 表
mysqldump -u root -p test main > test_main.tex
```

## 恢复至其他主机

sql 外导入：

```bash
mysql -u root -p < all.tex
```

sql 内导入：

```bash
source all.tex;
```

## 机对机搞基

```bash
mysqldump -u root -p | mysql -h 000.000.000.000
```
