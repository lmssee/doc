# 面向对象的实现

面向对象实现就是选择一种合适的面向对象的编程语言，如 C++、 Java
等，用选定的语言进行编码。当然也可以用非面向对象的编程语言进行编码。对详细设计所得到的软件系统的各对象类进行详尽的描述。将编写好的各个类代码模块根据类的相互关系集成。如果在实现阶段的工作过程中，发现分析或设计阶段隐藏的问题，需要及时返回相关的步骤作相应的调整，最终实现可运行的应用软件系统。

面向对象的软件开发方法对于结构化方法而言，有更好的直观性、可重用性和可扩展性及易于管理，当用户的需求发生变化时，也比较容易在以前开发的基础之上修改和完善，使得开发软件系统的过程更加高效、快捷。
