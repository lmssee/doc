# 文件传输

[fileZilla](https://filezilla-project.org/download.php#close) 貌似是一个不错的工具耶。开源且免费

如果可能的话，`scp` 也不错

- bye 退出
- cd path 更改目录
- lcd path （ local cd ）更改本地目录
- chgrp [-h] grp path 更改目录
- chmod [-h] mode path 更改目录权限 'mode'
- chown [-h] own path 将路径文件改所有者为 'own'
- copy oldPath newPath 复制文件
- cp oldPath newPath 复制文件
- df [-hi] [path] 显示当前目录或包含路径的文件系统
- exit 退出
- get [-afpR] remote [local] 下载文件到本地
- ln [-s] oldPath newPath 链接远程文件
- ls [-1afhlnrSt] [path] 查看当前目录
- lls [ls-options [path]] （ local ls ）显示本地目录列表
- mkdir path 创建远程目录
- lmkdir path （ local mkdir ）创建本地目录
- progress 展示进度条
- put [-afpR] local [remote] 上传文件
- pwd 展示当前目录
- lpwd （ local pwd ）展示本地当前目录
- quit 退出
- reget [-fpR] remote [local] 恢复本地下载
- rename oldPath newPath 重命名文件
- reput [-fpR] local [remote] 恢复上传文件
- rm path 删除远程文件
- rmdir path 删除远程目录
- symlink oldPath newPath 链接远程文件
- version 展示版本
- lumask umask 设置本地 umask
- !command 在本地 shell 使用 command
- ! 切换到本地 shell
- help 帮助文档
- ? 等同于 hel
