# screen

开启 screen （在 ssh 中直接输入 screen ， linux ），可以使已开启的服务进入后台，避免这边的操作导致原开启的服务被关掉。

| 命令              | 说明                                          |
| :---------------- | :-------------------------------------------- |
| screen            | 开启一个新会话（ ssh 界面）                   |
| screen -S {$name} | 开以一个新会话，并命名为特定的别名            |
| screen -ls        | 查看当前所有的会话                            |
| screen -r {$id}   | 切换会话，通过查看会话的命令，可查看会话的 id |
| screen -d {$id}   | 关闭已经在别的地进入 Attached 状态的子会话    |
| kill -9 {$id}     | 关闭会话，这会导致会话无法进入                |
| screen -wipe      | 清除被杀死的会话。                            |
| Ctrl + a + d      | 在会话内返回主会话                            |
