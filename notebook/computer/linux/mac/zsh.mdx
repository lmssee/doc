# 环境

编辑环境变量：

```zsh
vim ~/.zsh && resource ~./zshrc
```

## 坏掉了

如果不小心将其覆盖了：

```sh
 PATH=/bin:/usr/bin:/usr/local/bin:${PATH}export PATH
```

## 软链

有时候需要建立软链：

```sh
ln -s
```
