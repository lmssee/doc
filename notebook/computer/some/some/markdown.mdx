# markdown

轻量级的标记语言。允许人们使用更易读写的纯文本格式的编写文档。语言标准为 [common mark](https://commonmark.org/)

## 标题

## 一号标题

使用在下一行添加 ===== 或者在标题前添加 # 来标识一号标头。

```md
# 一号标题
```

## 二号标题

使用在下一行添加 ----- 或者在标题前添加 ## 来标识一号标头。

```md
## 二号标题
```

## 其他标题

几级标题就使用几个 # 来标识。

## 段落

直接书写即是段落。使用双空格及换行来换行表示。

## 斜体表示

```md
_斜体文本_

_斜体文本_
```

## 粗体表示

```md
**粗体文本**

**粗体文本**
```

## 粗斜体

```md
**_斜体文本_**

**_斜体文本_**
```

## 分割线

使用三个以上的 \_ 或者 - 来标识一个分割行：

```md
---
```

## 删除线

使用双波浪线来夹住需要标识删除的文本：

```md
~~被删的文本~~
```

## 下划线

使用 `<u>` 标识的文本：`</u>`

```md
<u>被下划的文本</u>
```

## 链接

使用 [链接文本](链接地址) 或者 <链接地址> 的形式来标识链接文本：

```md
[本站主页](https://lmssee.cn)
```

## 图片

开头一个 ! ，然后紧接着是 [art 属性文本] 最后是 (图片地址 “可选标题”)

```md
![art 我是一张图](https://www.mps.gov.cn/template/7569179/7569191.png'公安网备案标识')
```

## 列表

## 无序列表

使用 \* 或是 + 或是 - 来标识列表，需要在标志符与文本之间添加空格

## 有序列表

直接使用数字添加 . 来标识有序列表

## 列表潜逃

前面添加双空格，识别双空格后默认为二级列表。

## 区块

使用 > 标志符标识的文本为一个区块，标志符与文本之间需要有空格

## 代码

## 代码片段

使用 `` 包裹的文本：

```md
`我是代码片段`
```

## 代码域

使用四个空格代表代码区域

```md
    let a = 10;
    let b = 10;
    if (a == b) {
      console.log('a 等于 b 呦');
    }
```

或者使用 `en` 来包裹代码区域，也可以指定代码的语言：

```md
\`\`\`javascript
let a = 10;
let b = 10;
if (a == b) {
console.log('a 等于 b 呦');
}
\`\`\`
```

## 表格

使用 | 来分割格，使用 --- 来分割表头与其他部分：

## 对齐方式

- -: 来代表居右对齐
- :- 来代表居左对齐
- :-: 来代表居中对齐
