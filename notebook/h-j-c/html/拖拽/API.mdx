# 基础

## 启动拖拽

设置源对象的 draggable 属性，设置属性值为 true （ draggable="true" ），这样就可以启动拖拽功能。

a 、 img 标签默认开启了拖拽功能，但必须设置 href 。

## 编写处理函数

|           |            |                          |
| :-------- | :--------- | :----------------------- |
| dragstart | 源对象     | 开始                     |
| drag      | 源对象     | ing                      |
| dragenter | 经过的元素 | 源对象开始进入本经过元素 |
| dragover  | 经过的元素 | 源对象正在经过元素移动   |
| dragleave | 经过的元素 | 源对象离开经过的元素     |
| drop      | 目标元素   | 源对象被拖放进目标元素   |
| dragend   | 源对象     | 拖放操作结束             |
