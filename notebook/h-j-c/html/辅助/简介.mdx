# 辅助

ARIA 属性是由 WAI-ARIA （ Web Accessibility Initiative-Accessible Rich Internet Applications
）引入的，这些属性为存在视觉障碍的用户服务，可添加在任何 HTML 元素中，有两类定义方式： role 和 aria- （以 aria- 开头的属性）， role 属性定义了对象的通用类型（包括 radio 、
checkbox ， button 等）。 aria- 属性提供有关对象的特定信息，例如单选框或复选框的
checked 状态、按钮的 disabled 状态等。
