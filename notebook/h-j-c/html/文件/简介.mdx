# 访问文件域

在 HTML 4 中， file 控件只允许选择和提交一个文件， HTML 5 为 file 控件新添了 multiple 属性，允许用户在一个 file 控件内选择和提交多个文件。

为方便用户在脚本中访问这些将要提交的文件， HTML 5 新增了 FileList 和 file 对象。

- FileList ： 表示用户选择这些文件列表
- file ： 表示 file 控件内每一个被选择的文件对象。 FIleList 对象为这些 file 对象的列表，代表用户选择的所有文件。
