# 生成基本表格

要创建表格，必须从一个 `<table>` 标签开始。当然，要利用 `</table>` 标签结束表格。接下来需要 `<tr>` 标签。 `<tr>` 标签用于创建表格行，在该标签与

`</tr>` 封闭标签之间包含一个或多个信息单元格。要创建这些单独的单元格，可以使用 `<td>` 标签（ `<td>` 代表表格数据 [table data] ）。把表格信息放在
`<td>` 与 `</td>` 标签之间。单元格（
cell ）是一个矩形区域，可以包含任何文本、图像和 HTML 标签。表格中的每一行都包含至少一个单元格，一行中的多个单元格构成表格中的列。
