# 多媒体文件标签

`<embed>` 标签是 HTML 5 中的新标签，可以在页面中嵌入任何类型的文档。用户的机器上必须已经安装了能够正确显示文档内容的程序，一般常用于在网页中插入的多媒体格式可以是 Midi 、 Wav 、 AIFF
、 AU 、 MP3 等格式。

其中 url 为音频或视频文件及其路径，可以是相对路径或绝对路径。还可以通过 autostart 属性设置音频或视频文件是否在下载完之后就自动播放。

loop 属性规定音频或视频文件是否循环及循环次数。属性值为正整数值时，音频或视频文件的循环次数与正整数值相同；属性值为 true 时，音频或视频文件循环；属性值为 false 时，音频或视频文件不循环。例如：

| 属性      | 额                                                                                 |
| :-------- | :--------------------------------------------------------------------------------- |
| autostart | 定义音频或视频文件是否在下载完成之后自动播放                                       |
| hidden    | 规定控制面板是否显示，默认值为 no 。设置为 true 表示隐藏面板，设置为 no 则显示面板 |
| starttime | 设置音频或视频文件开始播放的时间，没有定义时从头播放。格式是" mm:ss "              |
| height    | 音频或视频文件的高度                                                               |
| width     | 音频或视频文件的宽度                                                               |
| type      | 定义音频或视频文件的 MIME 类型                                                     |
