# audio

HTML 5 支持 Ogg Vorbis 、 MP3、 Wav 等音频格式，用法如下：

```html
<audio src="samplesong.mp3" controls></audio>
```

- autoplay ：如果存在，则一加载完文件就播放它
- controls ：如果存在，则显示用于音频播放器的控件
- loop ：如果存在，则继续重复播放文件，直到手动停止它为止
- preload ：具有 3 个可能的值： none 、 auto 和 metadata 。如果不希望缓冲文件，就使用 none ；如果要缓冲文件，可使用 auto ；如果希望只缓冲文件的元数据，则使用
  metadata
- src: 多媒体地址

audio 可包含多个 `<source>` ，用来导入不同的音频文件，浏览器会自动找到一个支持的的格式进行播放。

可以在 audio 元素之间添加文本信息，用于不支持的浏览器的替换内容。
