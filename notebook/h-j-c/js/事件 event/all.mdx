# 所有

| 事件          | 当......时发生                     |
| :------------ | :--------------------------------- |
| onclick       | 点击                               |
| oncontextmenu | 在一个元素上单击鼠标右键以弹出菜单 |
| ondrag        | 元素被拖动                         |
| ondragend     | 在拖动操作的末端                   |
| ondragenter   | 当元素已被拖动到有效位置           |
| ondragleave   | 当元素离开有效拖放目标             |
| ondragover    | 当元素在有效元素上正在被拖动时     |
| ondragstart   | 当拖动元素拖动开始                 |
| ondrop        | 当被拖元素正在被拖动时             |
| ondblclick    | 双击                               |
| onmousedown   | 鼠标按下                           |
| onmouseenter  | 鼠标指针移到某元素上               |
| onmouseleave  | 鼠标指针离开该元素                 |
| onmousemove   | 鼠标指针在元素上移动               |
| onmouseover   | 鼠标指针移动到元素或其子元素上     |
| onmousewheel  | 当鼠标滚轮正在被滚动时             |
| onscroll      | 当元素                             |

| 事件       | 当......时发生         |
| :--------- | :--------------------- |
| onkeydown  | 用户正按下一个键       |
| onkeypress | 用户按下并释放一个按键 |
| onkeyup    | 用户释放一个按键       |

| 事件           | 当......时发生            |
| :------------- | :------------------------ |
| onafterprint   | 文件打印之后运行的脚本    |
| onbeforeprint  | 文件打印前运行的脚本      |
| onbeforeunload | 文件卸载之前运行的脚本    |
| onhaschange    | 当文档已改变时运行的脚本  |
| onerror        | 当加载文件出错            |
| onload         | 已经加载一个对象          |
| onresize       | 重新调整文档视图大小      |
| onscroll       | 滚动了元素的滚动条        |
| onmessage      | 当消息被触发时            |
| onoffline      | 文档离线时                |
| ononline       | 文档上线时                |
| onpagehide     | 窗口隐藏时                |
| onpageshow     | 窗口可见时                |
| onpopstate     | 窗口历史记录改变时        |
| onredo         | 文件执行撤销（ redo ）    |
| onstorage      | 在 Web Storage 区域更新后 |
| onundo         | 在文档执行 undo 时        |

| 事件          | 当......时发生                   |
| :------------ | :------------------------------- |
| onblur        | 一个元素失去焦点                 |
| onchange      | 内容、选取或选中的状态发生了变化 |
| oncontextmenu | 当上下文菜单被触发时             |
| onfocus       | 一个元素获得焦点                 |
| onforminput   | 当表单获得用户输入时             |
| oninput       | 当元素获得用户输入时             |
| oninvalid     | 当元素无效时                     |
| onreset       | 重置一个表单                     |
| onselect      | 用户选择了一些文本               |
| onsubmit      | 提交一个表单                     |

| 事件属性           | 描述                                               |
| :----------------- | :------------------------------------------------- |
| oncanplay          | 文件就绪可以播放                                   |
| oncanplaythrough   | 当媒介能够无需因缓冲而停止即可播放至结尾           |
| ondurationchange   | 当媒介长度改变时                                   |
| onemptied          | 当发生故障并且文件忽然不可用时                     |
| onended            | 媒介已到达结尾时                                   |
| onerror            | 文件加载期间发生错误                               |
| onloadeddata       | 当媒介元素已加载时                                 |
| onloadedmetadata   | 当元数据（分辨率和时长）被加载时                   |
| onloadstart        | 当文件开始加载且未实际加载任何数据前               |
| onpause            | 当媒介被用户或程序暂停时                           |
| onplay             | 当媒介已就绪可以播放时                             |
| onplaying          | 当媒介已经开始播放时                               |
| onprogress         | 当浏览器正在获取媒介数据时                         |
| onratechange       | 当回放速率改变时                                   |
| onreadystatechange | 就绪状态发生改变时                                 |
| onseeked           | 当 seeking 属性设置为 false （指示定位已结束）时   |
| onstalled          | 在浏览器无论何种原因未能取回媒介元素数据时         |
| onsuspend          | 当媒介数据完全加载之前无论何种原因终止取回媒介数据 |
| ontimeupdate       | 当播放位置改变时                                   |
| onvolumechange     | 每当音量改变时                                     |
