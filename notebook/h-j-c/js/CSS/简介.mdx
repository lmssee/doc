# 简介

DHTML 是 Dynamic
HTML 的缩写，即动态的 HTML 。它并不是某一门独立的语言，事实上任何可以实现 HTML 页面动态改变的方法都可以称为 DHTML 。最常用的方法就是将 JavaScript 、 HTML 和 CSS 相结合，这3个技术基本是相互独立的，但每种技术都为对方提供了 API ，实现相互操作。 CSS 也为 JavaScript 提供了 API ，使用这些 API 可以访问和设置 HTML 文档的 CSS 样式表。

使用 DHTML 技术，网页设计者可以创建出能够与用户交互并包含动态内容的页面，可以在网页被加载之后动态操纵网页上的所有元素，包括动态地隐藏或显示内容、修改样式定义、激活或者禁用元素以及为元素定位等。

CSS 脚本化是网页特效的基础， JavaScript 与 CSS 完美结合可以激活设计灵感。给页面带来惊艳的效果。 CSS 脚本化可以实现网页对象的显隐、变形、运动、特效、交互。
