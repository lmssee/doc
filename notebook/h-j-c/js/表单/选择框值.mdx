# 选择框的值

使用 select 和 option 元素可以创建选择框。 select 属于 HTMLSelectElement 类型，继承于 HTMLElement ，除了拥有表单字段公共属性外，还定义了下列属性和方法。

|                    |                                                                                                     |
| :----------------- | :-------------------------------------------------------------------------------------------------- |
| multiple           | 设置或返回是否可有多个被选择，等价于 `<select multiple>`                                            |
| selectedIndex      | 布尔值，设置或返回被选择的索引号，如果没有选中，则返回 -1 .对于支持选择的控件，只保存选中项的第一个 |
| size               | 设置或返回一次显示的选项，等价于<select size=\"4\">                                                 |
| length             | 返回选项的数目                                                                                      |
| options            | 返回包含所有选择的控件集合，即包含所有 option 元素的 HTMLCollection                                 |
| add(option,before) | 向控件中插入新 option 元素，其位置在 before 元素之前                                                |
| remove(index)      | 移除给定的选项                                                                                      |
