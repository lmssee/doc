# hi story

HTML 5 新增了一个 History API ，该 API 允许用户通过 JavaScript 管理浏览器的历史纪录，实现无刷新更改浏览器地址栏的链接地址，配合 History + Ajax 实现不刷新页面的跳转。

一般 History 与 Ajax 一起使用才有价值，应用中需掌握：

- 使用 Ajax 实现网页内容更新；
- 使用 History API 实现浏览器历史记录更新
- 使用 History API 实时跟踪浏览器导航响应，页面内容也随之更新
