# 创建数组

## 构造数组

- var a = new Array() 空数组
- var a = new Array(1,2,3,array,this) 初始化长度为 5 的数组
- var a = new Array(10) 初始化 length 为 10 的数组

## 数组直接量

- var a = [] 空数组
- var a = [1, 2, 3, array, this] 初始化长度 length 为 5 的数组

使用直接量定义数组，能够提高数组初始化运行效率。因此，在没有特殊的要求下，建议直接使用直接量定义数组。
