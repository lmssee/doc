# CSS 3 盒模型基础

CSS 3 规范新增 UI (User-interface)模块 [www.w3.org/TR/css3-ui](ttp://www.w3.org/TR/css3-ui)。该模块改善了传统的盒模型结构，增强了盒子构成要素的功能。

## 网页布局样式

根据网页宽度分类包括固定宽度（ px ）、弹性宽度（%）、液态宽度（ em ）和混合宽度。

根据页面版面分类包括一行一列、两行两列、两行三列、三行两列、三行三列等。

根据页面版面实现的方法分类包括流动布局、浮动布局和定位布局。

## 流动布局

流动布局是默认布局方式，所有元素在默认情况下，跟随文档流自上而下按顺序动态流动。

没有定义 position: absolute ; 或 position : fixed ； 属性，以及没有定义 float ： left/right ； 情况下，都是流动布局

当 position 属性设置为 relative 时，会尊随流动布局，但会在设置的相对定位的偏移量进行偏移。
