## 完善媒体特性与 Responsive 布局

CSS 3 媒体特性可以实现一种响应式（ Responsive ）布局，使布局可以根据用户的显示终端或设备特性选择对应的样式文件，从而在不同的分辨率或设备下具有不同的布局效果，特别是在移动端上实现一种理想的特性。
