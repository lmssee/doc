# attr()

在每个链接符号内插入 href 属性的值：

```css
attr(attribute-name)
```

## calc()

允许计算值：

```css
calc(expression)
```

超强大，支持 `+` 、`-` 、`*` 、`/`

## CSS 变量

var() 函数用以插入 CSS 变量的值

```css
var(name, value)
```

- name : 必需。变量名（以两条破折号开头）
- value ： 可选，回退值（未找到变量时应用）

该 CSS 变量也有局部和全局之分，全局就是用 ： root 选择器声明它。:root 选择器匹配文档的根元素。

使用 var() 更易于代码的阅读，也方便代码的更改。只改一处，全页受益。
