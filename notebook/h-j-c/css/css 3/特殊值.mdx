# 默认样式

新增属性值 ： initial ；能恢复到默认样式。

```css
p#text02 {
  color: initial;
  color: -moz-initial;
}
```

## 继承

inherit 表示继承祖先的属性值。

## 擦除设置

unset 为擦除用户的声明的值。如果有继承，这 unset 等同于 inherit 。

## all

all 表示所有的 css 属性，不包括 unicode-bidi 、 direction 属性。

## transparent

transparent 表示完全透明。

## currentColor

currentColor 表示 color 属性的值。
