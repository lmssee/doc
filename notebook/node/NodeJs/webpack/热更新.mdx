# 热更新 1.0 webpack-dev-server

安装：

```sh
npm install webpack-dev-serve --save-dev
```

安装指令 --save-dev 参数是将 webpack-dev-server 作为工程的 devDependencies （开发环境依赖）记录在 package.json 中。这样是因为
webpack-dev-serve 仅仅在本地开发时才可以用到。

在 webpack 配置文件中添加：

```json
{
  "devServer": {
    "static": "/dist"
  }
}
```

在 package.json 文件下 scripts 下添加：

```json
{
  "dev": "webpack-dev-server"
}
```
